# Reading Copilot - Claude Skill Pack

읽은 책의 흔적을 간직하고 싶은 당신을 위한 에이전틱 독서 코파일럿.  
Reading Copilot은 독서 과정에서 발견한 문구와 인사이트를 저장소에 차곡차곡 쌓아줍니다.  
어느새 나만의 지적 흥미로 가득한 소우주가 완성되어 있을 것! 💫

### Reading Copilot이란

이 책 저 책 헤멘 끝에 집어든 한 권의 책.  
단 10분. 길지 않은 시간이지만, 만족스러운 몰입이었다.  
생각이 뉴런을 타고 스쳐 지나가면서 다양한 생각들이 꽃피는 시간이었다.  
하지만 이내 유튜브와 인스타의 자극이 물밀듯 나의 관심을 앗아간다.  
그 순간, 10분의 보람된 수확도 휘발된다.

날아가버린 잠깐의 몰입의 시간을 붙잡아둘 수는 없을까?  
Reading Copilot은 이 질문에서 출발했다.  
Reading Copilot은 집중력이 희소한 나에게, 독서로부터 얻은 인사이트를 보관하고,  
창작이 필요한 순간 언제든 꺼내어줄 수 있는 똑똑이 사서다.

### 뭘 얻을 수 있느냐

책 사진(표지, 내지 등)을 Claude 에 업로드하고 `/rc` 를 호출하면 Obsidian 볼트에 :
- 책 노트 자동 생성
- 밑줄·하이라이트 텍스트 추출 및 저장
- 하이라이트가 쌓이면 관심사 자동 분석 → 태그 및 온톨로지 생성
- 책들 간 주제 연결 링크 자동 생성

### 이런 분들을 위해 만들었음

- **독서의 몰입 시간이 짧은 사람** 🫨
- **분명 책을 읽었지만, 내용을 설명할 수 없는 사람** 😶‍🌫️
- **완독이 어려운 사람** 🫠
- **지식에 대한 저장욕구가 있는 사람** 🧐

### 30초 시작하기

##### 준비물 🗃️
- **Claude Desktop / Claude Code**
  로컬 폴더 접근이 필요해서 Claude.ai는 아직 지원이 안됨 ㅠ 
  조금만 기다려주셈!
- **Obsidian**
  저장한 조각들을 관리하고 지식그래프로 시각화하는 뷰어
##### 설치하기 🛠️
1. 이 Repository에서 `reading-copilot_plugin.zip` 다운로드
2. Claude Desktop > Customize > Plugins > Add > Upload Plugin 탭에서 zip 파일 업로드
3. Claude 대화창에 `/rc_setup` 입력 - 노트 저장 경로를 알아서 설정해줌
##### 사용하기 ✏️
- Claude 대화창에 `/rc`를 입력하고 책 표지·내지 사진(밑줄 그은)을 올려보시라!  
    책별로 읽은 흔적을 알아서 차곡차곡 쌓아준다.
- 책을 읽으며 나눈 대화도 함께 아카이빙해준다.
- 책 뿐만 아니라 영화, 시리즈도 가능함 🎬
