# 제미나이와의 대화

내가 이걸 진정 왜 만드는 것일까, 정리할 필요가 있었다.

## 서비스의 페르소나 & 페인포인트

1. <span style="color:rgb(0, 176, 80)"><b>길 잃은 탐구자, The Disctracted Polymath</b></span>
	- ==사고 독립성 Low== / ==활용 동기 Introverted==
	- **특징** : 지적 욕구는 매우 높지만, 정보 과부하와 ADHD 성향으로 인해 관심사가 산만하게 흩어져 있음. 책을 사놓고 끝까지 읽지 못하는 건 물론, 목차만 읽고 만 책도 많음
	- **페인포인트** : 어제 읽은 내용이 오늘 기억나지 않고, 내 관심사가 어디로 흘러가는지 모르겠다
	- **해결점** : 단일 입력창으로 앱 활용 진입장벽을 낮추고, 대화 에이전트의 선제적 개입을 통해 몰입감 있는 독서를 지원한다. <span style="color:rgb(192, 0, 0)"><i>도구관리의 피로도가 없는 순수한 독서의 몰입 과정</i></span>
	  
2. <span style="color:rgb(0, 176, 80)"><b>지적으로 독립을 이루고 싶은 사람, The Sovereign Thinker</b></span>
	- ==사고 독립성 High== / ==활용 동기 Introverted==
	- **특징** : AI의 편리함을 인정하면서도, 나만의 사고 과정이 빅테크 기업의 데이터로 종속되는 것에 거부감을 느낌. 나만의 독자적인 사유의 성을 구축하고 싶어하며, 그것이 이 시대 생존의 길이라고 믿음
	- **페인포인트** : 내 지식의 계보는 내가 직접 소유한다, 모델과 서비스는 거둘 뿐
	- **해결점** : Vault-Centric 원칙을 통해 모든 인사이트를 Md 형태로 개인 저장소에 영구보존. *<span style="color:rgb(192, 0, 0)">파편회된 정보가 아닌 '연결된 지적 포트폴리오' 구축</span>*
	  
3. <span style="color:rgb(0, 176, 80)"><b>지식을 끊임없이 확장하고 연결하고 싶은 사람, Insight Architect</b></span>
	- ==사고 독립성 Low== / ==활용 동기 Extroverted==
	- **특징** : 새로운 지식이 기존의 다른 지식과 연결되는 지점에서 가치를 느끼는 사람. 서로 다른 분야에서 공통의 원리를 발견하거나, 이질적인 곳에서 나의 관심사와 링크되는 지점에서 행복감을 느낌
	- **페인포인트** : 책 한권을 읽으며 어찌어찌 정리한 포인트들, 휘발되는 나머지 제대로 사고의 확장을 경험하기 어렵다
	- **해결점** : 인사이트 통합 에이전트가 지식 노드간 연결을 통한 온톨로지 그래프를 작성해 줌. 나만의 지식 지도를 소유한다. <span style="color:rgb(192, 0, 0)"><i>플랫폼에 종속되지 않는 나의 영구적 지식 주권</i></span>
	  
4. <span style="color:rgb(0, 176, 80)"><b>질문하는 수행자, The Socratic Reader</b></span>
	- ==사고 독립성 High== / ==활용 동기 Extroverted==
	- **특징** : 단순히 읽는 행위를 넘어, 저자와 논쟁하고 질문하며 사고력을 키우고 싶어 함. 독서를 '학습'이 아닌 '사유의 훈련'으로 여김
	- **페인포인트** : 혼자 읽으면 생각의 틀에 갇힌다. 누군가 내 생각을 자극하고 날카로운 질문을 던져주길 바란다
	- **해결점** : 대화 에이전트의 소크라테스식 문답 기능을 통해 사고력을 외주화하지 않고, 오히려 촉진함. <span style="color:rgb(192, 0, 0)"><i>사고의 외주화가 아닌 사고의 증폭</i></span>
