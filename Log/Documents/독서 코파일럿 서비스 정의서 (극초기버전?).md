# 독서 코파일럿 서비스 정의서 ver 2.0 w/ Claude Code

## 서비스 개요

- **정의**

도서의 발견부터 완독 이후의 인사이트 아카이빙까지 전 과정을 지능적으로 보조하고 자동화하는 개인 맞춤형 독서 코파일럿 및 독서 온톨로지 시스템. 독서 과정에서 발견한 흥미로운 문구들에 기반하여 몰입감 있는 독서를 지원하여, 독서과정이 훑고 지나가는 휘발 과정이 아닌, 또렷한 지식 노드로 머릿속에 자리매김하도록 지원.

- **핵심**
    1. 사진 업로드를 통해 독서과정의 메타데이터 및 하이라이트/메모를 정제된 텍스트로 변환
    2. 사용자의 현재 관심사(프로젝트)와 책의 내용을 실시간으로 매핑하여 개인화된 시사점을 도출할 수 있도록 질의응답 (소크라테스식 문답)
    3. 메타데이터와 질의응답 과정으로부터 개인의 관심사를 추출하여 개인의 독서 온톨로지에 노드를 추가
    4. 모든 인풋과 아웃풋은 로컬 웹앱 단일 인터페이스에서 처리되며, 데이터는 Obsidian Vault에 저장되어 웹앱으로 실시간 불러옴
- **인터페이스 원칙**
    - 인풋(사진 업로드, 대화)과 아웃풋(결과 확인, 저장된 정보)이 **로컬 웹앱 하나**에서 통합 처리됨
    - Obsidian은 별도로 열어보는 뷰어가 아닌, 웹앱이 읽고 쓰는 **데이터베이스**로 작동
    - Obsidian MCP를 통해 웹앱과 Obsidian Vault가 실시간 연동됨

## 실행 프로세스

### 1️⃣ 책 정보 아카이빙

**어떤 책을 읽을지 탐색하는 과정**

- **Input** : 서점에서 관심 있는 책의 표지 사진을 촬영하여 로컬 웹앱에 업로드
- **Process** :
    - 업로드된 이미지를 Claude API (Vision)로 전송하여 책 표지 여부 식별
    - 책 표지로 판단될 경우 제목·저자 추출 후 웹 검색으로 메타정보 수집
    `제목`, `저자`, `출판연도`, `카테고리`, `저자 소개`, `책 소개`
    - 수집된 정보를 Obsidian MCP를 통해 Vault 내 지정 경로에 `[책 제목].md` 파일을 생성하여, 프론트매터에 메타정보를 저장
- **Output** : 웹앱 내 책 목록 패널에 즉시 표출 / Obsidian Vault에 자동 저장

### 2️⃣ 책 내부 정보 아카이빙

**하나의 책을 골라 읽기 시작**

- **Input** : 책을 읽는 과정에서 흥미롭게 읽은 페이지의 문구 이미지 (밑줄 혹은 하이라이트)를 웹앱에 업로드. 웹앱에서 현재 읽는 책을 선택한 상태에서 업로드하여 책 구분
- **Process** :
    - Claude API (Vision)로 촬영된 페이지의 전체 텍스트 + 하이라이트/밑줄 + 메모 식별
    - Obsidian MCP를 통해 해당 책 폴더 내 `highlights.md`에 누적 저장
- **Output** : 웹앱 내 해당 책의 하이라이트 패널에 즉시 표출 / Obsidian Vault에 자동 저장

### 3️⃣ 아카이빙된 정보에 기반한 개인화된 QA

**이 책에 대한 대화를 나눔으로써 독서 과정에서 얻은 인사이트를 내재화 (완독하지 않아도 됨)**

- **Input** : 사용자가 책을 읽고 느낀 점 / 궁금한 점에 대해 웹앱 채팅창에 질문
- **Process** :
    - Obsidian MCP를 통해 해당 책의 메타정보 + 저장된 하이라이트를 실시간으로 불러와 컨텍스트로 활용
    - 소크라테스식 문답으로 사용자의 생각을 이끌어내는 독서 퍼실리테이션 수행
    - 대화 기록과 도출된 인사이트를 Obsidian MCP를 통해 해당 책 폴더에 자동 저장
- **Output** : 웹앱 채팅창에서 실시간 대화 / 대화 종료 후 insights.md로 Obsidian에 자동 저장

## 서비스 아키텍처

### 인터페이스 구조

```
[로컬 웹앱 — 브라우저에서 접근]
┌──────────────────────────────────────────┐
│  📚 독서 코파일럿                         │
│                                          │
│  [책 선택 드롭다운 or 사이드바]           │
│                                          │
│  ┌──────────────┐  ┌───────────────────┐ │
│  │ 사진 업로드     │  │ 저장된 하이라이트     │ │
│  │ (드래그앤드롭)   │  │ (Obsidian에서     │ │
│  └──────────────┘  │  실시간 불러옴)      │ │
│                    └───────────────────┘ │
│  ┌──────────────────────────────────────┐│
│  │ 💬 대화창 (QA / 소크라테스식 문답)   ││
│  │                                      ││
│  │ [입력창]                    [전송]   ││
│  └──────────────────────────────────────┘│
└──────────────────────────────────────────┘
```

### 기술 스택

```
[로컬 웹앱]
    ↕ HTTP
[백엔드 서버 — 로컬 맥에서 실행]
    ├── Claude API (Vision + Text)
    │       ├── 1단계: 표지 인식 + 메타정보 추출
    │       ├── 2단계: OCR + 하이라이트 추출
    │       └── 3단계: QA 퍼실리테이션 + 인사이트 요약
    └── Obsidian MCP
            ├── 읽기: 저장된 노트를 웹앱으로 불러오기
            └── 쓰기: 처리 결과를 Vault에 자동 저장
```

### Obsidian Vault 구조

```
/Vault
  /Books
      /파친코
	      metadata.md          ← 책 기본 정보
      /생각에관한생각
        metadata.md          ← 책 기본 정보
        highlights.md        ← 누적 하이라이트
        qa_log.md            ← 대화 기록
        insights.md          ← 도출된 인사이트
```

### 과금 구조

| 항목 | 비용 |
| --- | --- |
